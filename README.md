# S.T.A.L.K.E.R. 2 — Інтерактивна Карта

## Структура проєкту

```
project/
├── index.html       ← головна сторінка
├── style.css        ← оформлення (темна тема)
├── map.js           ← логіка карти (SVG, drag, zoom, маркери)
├── charts.js        ← логіка списку та статистики
├── data.json        ← дані про всі об'єкти Зони
├── images/          ← сюди розмістити зображення об'єктів
└── stalker_map.png  ← фонове зображення карти (опційно)
```

## Запуск

Відкрити `index.html` у браузері Chrome або Firefox.

> Через завантаження `data.json` через fetch() потрібен локальний сервер:
> ```
> npx serve .
> ```
> або у VS Code: розширення **Live Server** → Right Click → "Open with Live Server"

## Функції карти

- Інтерактивна SVG-карта з маркерами
- Фільтри по типу (локація / аномалія / артефакт)
- Пошук по назві
- Масштабування (скрол, кнопки +/−)
- Додавання своїх міток
- Перетягування карти
- Тултіп з деталями при наведенні
- Вкладка "Список" з картками та пошуком
- Вкладка "Статистика" з графіками

## Структура data.json

```json
{
  "id": 1,
  "name": "Назва мітки",
  "type": "Тип мітки",
  "zone": "Назва зони",
  "description": "Опис",
  "coords": [320, 150]
}

```

## Додавання нових об'єктів

Додайте новий запис у відповідний масив у `data.json`. Координати `[x, y]` відповідають системі координат SVG (viewBox 0 0 900 800).

## Браузери

Перевірено: Chrome 120+, Firefox 121+
